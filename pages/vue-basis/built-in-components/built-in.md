# Built-in components

Vue propose des [composants natifs](https://vuejs.org/api/built-in-components.html), que l'on peut utiliser dans nos applications :
- [Transition](https://vuejs.org/guide/built-ins/transition)
- [Keep-alive](https://vuejs.org/guide/built-ins/keep-alive)
- [Teleport](https://vuejs.org/guide/built-ins/teleport)
- [Suspense](https://vuejs.org/guide/built-ins/suspense.html)
